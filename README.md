# CEF4Delphi
CEF4Delphi is an open source project created by Salvador DÃ­az Fau to embed Chromium-based browsers in applications made with Delphi.

CEF4Delphi is based on DCEF3, made by Henri Gourvest. The original license of DCEF3 still applies to CEF4Delphi. Read the license terms in the first lines of any *.pas file.

CEF4Delphi uses CEF3.2883.1552.g88ff29a which includes Chromium 55.0.2883.87.
